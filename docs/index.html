About these figures

These are estimates for the server powering BartlebyGPT, which as currently deployed on an RTX A4000 in a data center in Washington State. The server can generate up to 15 simultaneous conversations, so these figures are the cost to serve all users.

Watts is the estimated total facility draw for this server — GPU, CPU, RAM, storage, fans, and the data center's power distribution and cooling overhead (PUE ~1.15). Idle reflects background system load; active reflects full GPU utilization during inference.

gCO₂/hr is the direct carbon cost of the electricity consumed — wattage × grid carbon intensity. It does not include manufacturing, shipping, or end-of-life costs for the hardware. This server runs on hydroelectric power in Washington State, so the grid intensity is very low (~0.030 gCO₂/Wh vs. the US average of ~0.386).

$/hr is the cost to rent this GPU from the data center — not a per-query charge, not electricity, not amortized hardware. The server accrues this cost whenever it is running, idle or not.

All three parameters are adjustable under Advanced.
